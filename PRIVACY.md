# 隐私说明

paste3 是剪贴板历史工具，会处理高度敏感的本地数据。本文件说明当前实现的隐私边界。

## 会读取什么

paste3 会监听 macOS 系统剪贴板变化，并尝试识别以下内容：

- 文本、链接和命令。
- 图片。
- 文件 URL。
- 富文本、HTML。
- 其他可保存的通用剪贴板数据。

每条历史会保存内容摘要、搜索文本、来源应用名称、来源 bundle identifier、内容 hash、字节大小，以及必要时的原始 payload 数据。

## 数据保存在哪里

数据通过 SwiftData 保存在本机应用存储中。当前版本没有服务端、云同步、账号系统、远程数据库或远程分析。

## 不会做什么

- 不上传剪贴板内容。
- 不向第三方服务发送剪贴板数据。
- 不做遥测或行为分析。
- 不在仓库中内置远程 API key。

## 辅助功能权限

辅助功能权限只用于自动粘贴：当用户在 quick panel 中选择历史项并请求粘贴时，paste3 会切回原前台应用并模拟 `Cmd+V`。

如果不授予辅助功能权限，paste3 仍可复制历史项到系统剪贴板，但不会自动向目标应用发送粘贴快捷键。

## 用户控制

用户可以从菜单中清空历史记录，也可以删除单条历史记录。当前默认保留最近 1 年历史，超过保留时间后会自动裁剪较旧内容；也可以在设置中改为 1-6 天、1-3 周、1-11 个月、1-3 年或永久保留。

## 已知限制

- 当前版本还没有按应用排除、敏感内容自动忽略、暂停监听或加密存储。
- 如果 macOS 应用身份、Bundle ID 或签名发生变化，系统可能把它当作新应用，已有权限和本地数据路径也可能随之变化。

这些限制会影响隐私体验，后续应优先补齐。
